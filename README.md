Rate Limiter Scala
-----------
Rate limiter implemented with scala.

Development
-----------
There should be "public/hoteldb.csv" file present in project where
columns -> {CITY,HOTELID,ROOM,PRICE}

Component version:

    java -> 1.8
    scala -> 2.13.8
    sbt -> 1.5.2

To start the server, run:
    
    sbt run

To trigger functional tests, run:
    
    sbt test
